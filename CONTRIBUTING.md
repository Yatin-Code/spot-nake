# Contributing to SpotNake

Thank you for your interest in contributing to SpotNake! We welcome contributions to improve the bot's features, architecture, and overall stability.

## Contribution Guidelines

1. **Fork the Repository:** Create a personal fork of the repository on GitHub.
2. **Create a Feature Branch:** Choose a descriptive branch name:
   ```bash
   git checkout -b feature/NewCapability
   ```
3. **Ensure Code Quality:** Ensure all tests pass and code complies with standard Python PEP 8 formatting.
4. **Commit Changes:** Write clear, concise commit messages:
   ```bash
   git commit -m 'Add NewCapability'
   ```
5. **Push & Pull Request:** Push the changes to your fork and submit a Pull Request against the main branch.
   ```bash
   git push origin feature/NewCapability
   ```

Please restrict pull requests to functional improvements, bug fixes, or architecture optimizations.
